# 生物学的検定法シミュレーター
## 概要
- 松本歯科大学の学生実習で行っていた生物学的検定法の実習項目を
  コンピューター上で再現したシミュレーターです


## 実行方法について
- [このページ](https://toshi-ara.github.io/simla-ts/sim_bioassay.html)
  にアクセスして下さい


## 実習手順について
### シミュレーターの元となった動物実験の手順
1. 実習準備
    - リドカインの用量および各群の匹数を決定する
    - リドカイン溶液を用意する
1. 実習
    - マウスに指定された用量になるようにリドカインを腹腔内注射する
    - マウスを45°に傾けた金網に置く
    - 金網を15cm以上すべり落ちた場合にけいれんが生じたと判断する。
      また、体動が見られなくなった場合に死亡したと判断する
1. TD<sub>50</sub>およびLD<sub>50</sub>の算出
    - プロビット回帰分析で算出する

### シミュレーターの使用方法について
#### 初期画面
1. 画面下部のプルダウンメニューで言語を選択する（以下、日本語画面での説明）
1. 実験ボタンを押すと実験画面に移動する
1. TD<sub>50</sub>/LD<sub>50</sub>算出ボタンを押すと算出画面に移動する
1. 設定ボタンを押すと設定画面に移動する
    - 動画再生のオンオフ
    - 用量の設定（td<sub>50</sub>用およびld<sub>50</sub>用）

#### 実験画面
1. マウスの体重が表示されるため、使用する薬物濃度をプルダウンメニューから選択し、
   投与量を算出する
1. 必要に応じて判定時間（最大観測時間）をプルダウンメニューから選択する
1. 薬物投与ボタンを押すと動画が再生されて結果が表示される
1. 新規実験を押してから次の実験を行う
1. 戻るボタンを押すと初期画面に戻る

#### TD<sub>50</sub>/LD<sub>50</sub>算出画面
1. "合計"に各群の匹数を、"例数"に反応を示した匹数を入力する
1. 算出ボタンを押すと結果および表、グラフが表示される
1. 戻るボタンを押すと初期画面に戻る

#### 設定画面
1. 動画再生のオンオフをプルダウンメニューから選択する
1. 用量の設定（TD<sub>50</sub>用およびLD<sub>50</sub>用）を選択する
1. 変更ボタンを押すと反映され、キャンセルボタンを押すと反映されずに初期画面に戻る


### このシミュレーターの注意点について
- このシミュレーターは実際の実習結果をもとにモデル化したものですが、
  必ずしも動物実験と同じ結果が得られるとは限りません

## 環境構築について（教員用）
- MIT Licenseで公開しているため商用・非商用に限らず使用・改変可能です
- 以下の方法のいずれかで実習環境を構築します
    1. GitHub上に公開したページにアクセスする方法
        - https://toshi-ara.github.io/sim_bioassay
        - 最も簡単ですが、学生がアクセスしやすいページ（教室独自のページなど）に
          このリンクを貼っておくといいと思います
    1. 自前のwebサーバに設置する場合（**詳しい方に相談して下さい**）
        - このページの`Code`ボタンをクリックすると
          `Download ZIP`と表示されるためクリックし、
          `sim-bioassay-main.zip`という名前のファイルを保存します
        - zipファイルを解凍します
        - `docs`フォルダの中身をwebサーバの適切な場所に設置します
          （適宜フォルダ名を変更して下さい）
        - ウェブサーバにアクセスするとシミュレーターを使用できます

### ビルド方法
- npm（JavaScriptのパッケージ管理システム）が必要です
- npmをインストールした状態で以下のコマンドを実行します

```bash
git clone https://github.com/toshi-ara/sim-bioassay.git
# git clone git@github.com:toshi-ara/sim-bioassay.git  # sshを使用する場合
cd sim-bioassay

npm install
npm run build
```
